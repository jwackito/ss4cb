Strategy Design Pattern (Calculator Implementation)

# Author

# Algorithm description

The Strategy Pattern is a behavioral design pattern that lets you define a family of algorithms, put each of them into a separate class, and make their objects interchangeable. In this implementation, a `Calculator` context can switch between different mathematical operations (`Addition`, `Multiplication`) at runtime without changing its core logic.
